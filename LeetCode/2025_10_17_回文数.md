
解题使用的是双指针法（Python）：
```
class Solution(object):

    def isPalindrome(self, x):

        """

        :type x: int

        :rtype: bool

        """
        s = str(x)
        left = 0
        right = len(s) - 1

        while right > left:
            if s[left] != s[right]:
                break
            left += 1
            right -= 1

        if left >= right:
            return True
        else:
            return False
```

需要注意的是`while`循环判断条件中，若写作`while right != left`则容易导致指针越界，比如数字`11`
