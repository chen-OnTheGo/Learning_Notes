```
static int binarySearch(String[] sorts, String x, int lo, int hi){
	int m = (lo + hi) / 2;
	int cmp = x.compareTo(sorted[m]);
	if(cmp < 0){
		return binarySearch(sorted, x, lo, m - 1);
	}
	else if(cmp > 0){
		return binarySearch(sorted, x, m + 1, hi);
	}
	else
		return m;
}
```
