
## 概念

定义： 组件上注册的一些自定义属性。
作用：向子组件传递数据。


## 特点

- 可以传递**任意数量**、**任意类型** 的 `prop`。


## `prop` 校验

当我们传递 `prop` 时，为了避免传递错误，需要为组件的 `prop` 指定验证要求，若不符合要求要有错误提示，便于我们快速发现错误。

### 语法

1. 类型校验（最常用）
```
props: {
	校验的属性名: 类型
}
```
2. 非空校验
3. 默认值
4. 自定义校验（可自定义要求）

```
props: {
	校验的属性名: {
		type: 类型,
		required: true,
		default: 默认值,
		validator (value){
			//自定义校验逻辑
			return 是否通过校验
		}
	}
}
```
