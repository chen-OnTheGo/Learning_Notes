
## 本质

`:属性名` 和 `@update:属性名` 合写。
e.g：`:visible.sync="isShow"` 等效于：
```
:visible="isShow"
@update:visible="isShow = $event"
```


## 作用

可实现子组件和父组件数据的双向绑定，简化代码。

## 特点

`prop` 属性名，可以自定义，不一定必须为`value`。（与 `v-model` 的区别之一）
